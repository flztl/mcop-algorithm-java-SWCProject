# MCOP - Marketing Campaign Optimization Project

## Description
This project solves the Marketing Campaign Optimization Problem (MCOP) using various algorithms such as:
- Greedy Algorithm
- Dynamic Programming
- Backtracking
- Divide and Conquer

It also includes:
- Insertion Sort
- Binary Search
- Min-Heap
- Splay Tree

## Language Used
Java

## How to Run
1. Compile the Java file
2. Run the main method
